# TravelTracker

A web application for creating and managing a wishlist of travel destinations. Users can search for popular tourist places by keyword and geolocation using the Foursquare Places API. The application allows adding places to a wishlist, viewing detailed information (ratings, types, photos, tips/reviews), and implements a 10-minute cache for API requests. Saved places persist in local storage across page reloads.

## Installation

Install dependencies:

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm start
```

The application will be available at `http://localhost:4200/`
